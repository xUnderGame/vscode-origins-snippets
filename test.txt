                <h1 id="power_types">Power Types</h1>
<p>Power Types are what grants functionality to your origins! Each power has a type, specified with
a <code>type</code> field in the JSON. Which type a power is defines which other fields it requires and supports.</p>
<div class="admonition note">
<p class="admonition-title">Note</p>
<p>Unless stated otherwise, each power type supports a <code>condition</code> field that can check for <a href="../entity_condition_types/">Entity Condition Types</a>. See <a href="../../json/power/">Power JSON</a> for more details.</p>
</div>
